Markup Review

Personal website of Finnish web designer Aleksi Liukkonen.

A lang attribute should probably be added indicating that the site is in Finnish. The hgroup element is used here, enclosing a h1 and h2, which are both wrapped in an a. Some WAI-ARIA roles are also used throughout the site.
